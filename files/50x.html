Bartendro start up error

Bartendro failed to start up. :(

Check the following items which could cause Bartendro to fail to start:

  • System clock is incorrect: If the system doesn't have a valid date/time it may fail to start.
  • Functioning router: Make sure that the router is connected and has power and that the battery holder has a good battery.
  • Functioning pumps: Make sure that all dispensers are properly connected and functioning. Consider disconnecting one dispenser at a time to idenitfy possible troublesome dispenser. Make sure to disconnect power before connecting/disconnecting dispensers!
